# XIRASM 中文文档

XIRASM 是一个面向 x86、x86-64、RV32 与 RV64 的现代汇编器。除了直接编写处理器指令，它还提供完整的编译期语言，用于计算、控制流、代码与数据生成、模块复用和小型 DSL。
