# Win32 Metadata Notice

The generated catalogs in this directory are derived from the Win32 metadata
schema published by the Microsoft
[win32metadata](https://github.com/microsoft/win32metadata) project and its
partitioned [win32json](https://github.com/marlersoft/win32json) distribution.
The metadata license is reproduced in `LICENSE.win32metadata`.

The metadata interpretation and namespace partitioning were cross-checked
against the open-source
[zigwin32](https://github.com/marlersoft/zigwin32) bindings and
[zigwin32gen](https://github.com/marlersoft/zigwin32gen) generator. No generated
Zig bindings are included. The zigwin32 license notice is reproduced in
`LICENSE.zigwin32`.

The exact metadata version and source digest used for this catalog are recorded
in `manifest.json`.
